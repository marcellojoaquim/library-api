# Library API Project

# About this project

ApiRest developed in java with spring boot, in this project you will have api for books 
and for managing book loans, you will also have a previous configuration for Spring Admin and travis-ci.

# Book API
-Base URL: api/books

## Methods HTTP
- Create
- Update 
- Get
- Get loans by book

# Loans API
-Base URL: api/loans

## Methods HTTP
- Create
- Find
- Get by id


# Technologies Used
- Java 17
- Maven
- Spring Boot
- Spring Admin
- H2 DataBase
- Postman
